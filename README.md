# 地球物理数字信号处理技术 配套代码

本仓库是科学出版社教材《地球物理数字信号处理技术》的配套开源代码库，包含了书中所有核心算法的实现，以及典型的地球物理正演、反演和信号处理案例。欢迎各位同行、同学批评指正！

## 📚 项目简介
随着全球能源需求的持续增长，地球物理勘探技术在油气与矿产资源等领域发挥着日益重要的作用。其中，地震勘探作为核心手段，依赖数字信号处理技术以获取高质量的地下结构信息。

本代码库配套教材，系统实现了从基础信号处理到前沿深度学习反演的全流程算法，旨在帮助读者将理论知识与实践应用相结合，更好地掌握地震信号处理的核心原理与技术。

## 📁 代码结构
本仓库按教材章节和功能模块组织，各目录对应书中的核心内容：

```
├── 基于TensorFlow平台的DnCNN代码    # 深度学习去噪算法实现
├── 层状介质大地电磁测深正演        # 大地电磁测深正演模拟
├── 球体的重力异常正演              # 球体模型重力异常正演
├── 球体磁异常正演                  # 球体模型磁异常正演
├── 简单储层模型的地震褶积正演      # 褶积模型地震响应正演
└── 附录程序                        # 教材附录配套代码
```

## 🛠️ 主要功能
1.  **基础信号处理**：实现了傅里叶变换、褶积、滤波、时频谱分析等经典算法
2.  **正演模拟**：包含重力、磁法、大地电磁、地震等多种地球物理场的正演代码
3.  **反演方法**：提供了波阻抗反演、贝叶斯反演等传统方法，以及基于深度学习的智能反演实现
4.  **深度学习应用**：包含基于概率扩散模型、循环神经网络和DnCNN等前沿AI算法在地震信号处理中的应用
